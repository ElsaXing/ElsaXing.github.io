Game Design Part 1

每个玩游戏的人都有一个颗做游戏心？

最近重新开始学被我丢弃了好久的unity，从最基础的开始学，还没什么好总结的。今天试了一下coursera上的游戏设计导论，想着把笔记什么的写在这边吧。第一周只讲了一些很基础的设定。

第一周课程笔记与想法:

Game:

  • Goal: all games need a clear goal.

  • Chance and Skill: the balance between chance and skill

    所有的游戏都需要有一个明确的目标，战胜对手？收集多少东西？完成多少任务？没有目标会让玩家失去继续游戏的动力。同时，游戏需要一个升级的过程，难度变大，奖励变高，才可以刺激玩家继续前进。
    有些游戏是靠运气进行的，比如老虎机？还有一些是靠不断练习的经验技巧进行的，比如棋类游戏。好的游戏需要做到运气与技巧的平衡。

Game design document:

  • Description of the game
  • Description of the gameplay involved
  • Description of the game’s goal, puzzles, and characters
  • Illustrations, maps, and story lines

  • Sometimes, Technical information

    游戏的设计需要被统一记录，把所有的想法和设定都记录下来，不仅方便自己查询，有团队合作的时候也方便大家理解和讨论。

Start a game

  • An interesting idea.

开始做一个游戏，你需要一个简单有趣想法，它不需要是完美的，有趣就够了，没有必要浪费大把的时间在开始之前。确定一个想法，然后发展它。你会在过程中发现它很棒，或者很糟糕，但，你不会在一开始就知道的，所以，have a try！之后你可以选择继续发展它，或者，换一个新想法，然后不断积累经验。

如果实在没有什么想法，问问周围的人，总会有很多好或者不好的建议，或者，观察你的周围，你可以从任何东西中得到启发，音乐？电影？宠物？但是，请尽量避开那些你已经接触过的游戏，你需要一个有它独特之处的想法。
